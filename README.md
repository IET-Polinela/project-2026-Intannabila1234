### Catatan akses

Bootstrap script:

- Untuk membuat user testing dan contoh laporan, jalankan script bootstrap:

```bash
python scripts/bootstrap_lab10.py
```

- Untuk keamanan, token JWT tidak ditampilkan ke stdout secara default. Jika Anda perlu melihat token untuk debugging, jalankan:

```bash
python scripts/bootstrap_lab10.py --show-tokens
```

Catatan keamanan: Jangan bagikan token yang dicetak ke publik. Flag `--show-tokens` hanya untuk debugging lokal.
